Example: Send a Reading

Example JSON that a device can POST to /api/readings:

{
  "thing_id": "station-001",
  "timestamp": "2025-12-10T13:45:00Z",
  "temperature": 22.4,
  "humidity": 56.1,
  "pressure": 1013.2,
  "battery": 92
}

cURL example:

curl -X POST -H "Content-Type: application/json" \
  -d '@reading.json' http://localhost:5000/api/readings
